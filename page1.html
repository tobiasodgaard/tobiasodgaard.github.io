A Virtual Playground in Space

Master Thesis Project, M.Sc. Games Technology, IT University of Copenhagen 2020

The virtual playground is a project investigating play. Set in a spaceship heading to Mars, the players will have to entertain themselves using the different toys placed around in the spaceship.

Turn the gravity on and off, grow plants, build structures and befriend the onboard robot.

The main purpose of this project is to investigate how players can be motivated intrinsically, that is, without points, levels, missions etc.

Specifications: Developed in Unity & FMOD

Roles: Game designer, main programmer and sound designer

Thesis Group: Rasmus Emil Odgaard, Adam Ryde Ankarfeldt & Nikolaj Schildt Christensen

Supervisor: Miguel Angel Sicart
